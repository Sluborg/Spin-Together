# Credits & asset licenses

All assets in this project are **original or CC0**. No Luck Be A Landlord art, names, text, or
sounds are used — this project borrows *mechanics* only.

Each asset added by the `asset-fetcher` agent is logged here with its source URL and license.
Primary source: **Kenney.nl** (CC0). Fallbacks: **OpenGameArt** and **Freesound**, filtered to
CC0 only.

## Symbols (icons)
_Pixel-art at 16px (Kenney "Tiny" family style), upscaled nearest-neighbor + normalized to
128×128. **Frame-free tiles preferred** (Kenney "item" tiles carry an inventory-slot frame, so
the coin + gem are original CC0 pixel art in the same style). All CC0 — no attribution required.
Full rationale + rejected tiles in `docs/assets/pack-map.md`._
